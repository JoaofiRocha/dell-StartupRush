


FrontEnd{
  1. Dependencias
    npm install react-brackets
    npm install antd
    npm install react-router-dom
    
  2. Rodar FrontEnd
    cd frontend
    npm start
}

      com ./mvnw spring-boot:run ou pela propria ide
}

Com ambos rodando abrir localhost:3000 e utilizar a aplicacao
